01 — The Problem

The same event. Three completely different stories.

When a missile strikes a city, Russian state media, Ukrainian outlets, and Western sources describe the same event in entirely different ways — different facts, different framing, different omissions.

The Core Problem

Readers have no tool to understand these differences systematically. They either consume one narrative and believe it — or feel overwhelmed and disengage entirely. This fuels polarization, enables disinformation, and erodes trust in journalism globally.

This is not a new problem — but it has become dramatically more acute. The Russia-Ukraine war is the most documented information war in history. The same dynamic plays out in the Israel-Palestine conflict, the US-China rivalry, every contested election. Billions of people are exposed daily to manipulated narratives with no tools to detect the manipulation.

Existing tools fall short

Tool What it does What it misses
Ground News Left vs. right bias in US media No geopolitical narratives, no AI explanation
AllSides US political spectrum comparison US-only, no international narratives
GDELT Raw event data at scale No product for general audiences
NewsGuard Source credibility ratings No cross-narrative comparison

The gap: no tool combines geopolitical narrative comparison with AI-generated explanation of divergence, accessible to general audiences in multiple languages.

02 — The Solution

NarrativeScope: one event, multiple realities, AI explanation.

NarrativeScope automatically identifies the same event across 50+ sources from different media ecosystems, clusters related articles, and generates a structured AI analysis showing exactly where — and why — narratives diverge.

How it works

RSS feeds from tagged sources → AI embedding & event clustering → filter: 3+ sources covering same event → Gemini analysis: facts agreed, emphasis, omissions → multilingual output to readers.

Live example: Zaporizhzhia NPP blackout, May 27, 2026

Same event — three narratives
Russian Media (TASS · RT)

IAEA confirmed the blackout. Cause: Ukrainian military strike on Energodar. Russia framed as victim protecting nuclear infrastructure. Grossi's statement used to validate severity.

Ukrainian Media (Ukrinform)

Blackout confirmed, IAEA investigating. Framed as consequence of Russian occupation. 317 frontline engagements reported. All Russian actions described as "repelled."

Western Media (BBC · Reuters)

IAEA confirmed. Cause "under investigation." No definitive attribution. Context: plant under Russian control since 2022. Focus on IAEA process, not blame.

✦ Gemini AI — Narrative Analysis

All three ecosystems cited the same IAEA statement — a rare convergence. But they diverge sharply on causation. Russia controls the plant yet frames itself as victim. Ukraine reports intense combat yet acknowledges no territorial changes. Western media — limited by access — anchors on institutional sources. Key divergence: occupier-vs-defender framing applied to the same physical infrastructure.

Key features

  • Automatic event clustering using Vertex AI embeddings — no manual curation
  • Source taxonomy: Russian state / Ukrainian / Western / independent Russian / Chinese / other
  • AI comparison: shared facts, framing differences, what each side omits
  • Fact verification: cross-source claim checking with confidence scores
  • Multilingual: summaries available in EN, RU, ZH, DE, FR, UA
  • Disinformation flagging: claims refuted across 3+ independent sources

Cost efficiency by design

The system is built to minimize AI calls. Only events covered by 3+ independent sources trigger analysis. Headlines and 2-sentence summaries are translated (Gemini Flash); full text only on user request (lazy translation, cached 30 days). Estimated cost at MVP scale: $10–25/month.
